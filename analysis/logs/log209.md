# Log209 — SmallFastDronev1 Outdoor Hover (Tuning Verification)

**Date:** 2025-02-14
**Vehicle:** SmallFastDronev1, BF_X quad
**Mode:** Loiter
**Duration:** ~49s flight (arm 48.7s, disarm 101.4s)
**Purpose:** Verify gain reductions fix 8-10 Hz roll oscillation from log208

## Parameter Changes from Log208

| Parameter | Log208 | Log209 | Change | Rationale |
|-----------|--------|--------|--------|-----------|
| ATC_ANG_RLL_P | 27.25 | **20.0** | -27% | Break coupled angle-rate limit cycle |
| ATC_ANG_PIT_P | 31.54 | **27.0** | -14% | Reduce pitch coupling gain |
| ATC_RAT_RLL_P | 0.0562 | **0.048** | -15% | Reduce roll rate loop gain |
| ATC_RAT_RLL_I | 0.0562 | **0.048** | -15% | Match P |
| MOT_THST_HOVER | 0.125 | **0.095** | -24% | Reduce feedforward (still too high) |

Unchanged: RAT_PIT_P/I/D, RAT_RLL_D, all FLTE/FLTD/SMAX, ANG_YAW_P.

## Roll Oscillation — Eliminated

See [outdoor tuning topic](../topics/outdoor_tuning_sfdv1.md#verification-log209) for full analysis.

Summary: 8-10 Hz limit cycle from log208 is gone. Roll overshoot ratio dropped from
1.52 to 0.82 (below 1.0 = rate loop no longer amplifying). Dominant frequency shifted
from 8-10 Hz to 3 Hz (normal flight dynamics). 7-12 Hz band power reduced from 55%
to 18%. D-term no longer shows coherent oscillation structure.
