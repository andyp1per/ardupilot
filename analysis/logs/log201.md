## Yaw Inconsistency — Root Cause Found

**Symptom:** EKF yaw inconsistency error during flight, 24 degrees yaw divergence
between Core 0 and Core 1.

**Root cause chain:**
1. Two EKF cores on different IMUs (EK3_IMU_MASK=3) sharing one compass
2. COMPASS_MOTCT=0 — no motor interference compensation
3. EK3_MAG_CAL=4 (ALWAYS) learning body offsets during flight
4. Motor interference measured at -236 mG/throttle on X, -119 mG/throttle on Y
5. Core 1 MX body offset ran from +2 to -58 mG (runaway learning)
6. Core 0 MX stayed stable at -5 to +6 mG
7. 56 mG MX divergence between cores -> 24 degree yaw split

**Why Core 1 diverges more than Core 0:**
Different IMU noise characteristics cause different Kalman gain tuning, making
Core 1 more susceptible to compass innovation errors during motor interference.

## Recommendations from This Log

1. Set COMPASS_MOTCT=2 (current-based motor compensation) — fix the root cause
2. Set EK3_MAG_CAL=7 (new mode) — protect takeoff phase from interference
3. Set EK3_RNG_USE_HGT=-1 — fix terrain offset feedback loop
4. Reduce ATC_ANG_YAW_P from 17.8 to 4.5-6.0 — reduce yaw twitchiness
