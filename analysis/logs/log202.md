# Log202 — SmallFastDronev1 Indoor Loiter

**Date:** 2025-02-14
**Vehicle:** SmallFastDronev1, BF_X quad
**Mode:** Loiter
**Duration:** 82.6 seconds
**Purpose:** Test loiter performance with updated parameters

## Key Parameters

Same as log201 (EK3_MAG_CAL=4, COMPASS_MOTCT=0, EK3_RNG_USE_HGT=3.0)

## Altitude Hold

- BAlt mean: 0.39m (much improved over log198 which crashed)
- TCAL working through 13.6C temperature drop
- No crash — the PSC_POSZ_P=1.0 and TKOFF_GNDEFF_TMO=3.0 changes helped

## Yaw Behavior

- Yaw divergence between cores: ~3 degrees (much better than log201's 24 degrees)
- In-flight yaw alignment at t=57s caused 18 degree heading snap
- Second correction at t=63s (12 degrees)
- These heading jumps are visible to the pilot

**Yaw twitchiness while hovering (sticks centered):**
1. In-flight yaw alignment jumps at t=57-63s
2. ATC_ANG_YAW_P=17.8 (4x default of 4.5)
   - With P=17.8, a 0.2 degree heading error produces 3.6 deg/s yaw rate demand
   - This chases compass noise aggressively, causing visible twitchiness
   - Recommend reducing to 4.5-6.0

## Terrain Offset

Still noisy at +/-1.7m due to EK3_RNG_USE_HGT=3.0 — the terrain offset feedback loop
is still active. This confirms the need for EK3_RNG_USE_HGT=-1.
